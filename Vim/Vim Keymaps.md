### Motions
	H - Start of the line
	L - End of the line
### Deleting
	<C-h> : Delete one letter before cursor
	<C-k> : Delete from cursor to the end of line
	[Edit Mode] <C-w> : Delete one word
	x : delete current cursor
### Indentation
	>> Indenting
	<< Outdent
